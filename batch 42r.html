What is Browse?

A browser is a software application used to access and view websites on the internet.

Examples:

Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge.

What is the Internet?

The internet is a global network of connected computers that allows people to share information and communicate from anywhere in the world.

Examples:

web, email, social media, and more.

What is URL?

A URL (Uniform Resource Locator) is the complete web address used to access a specific resource (like a webpage, image, or video) on the internet.

Examples:

URL: https://WWW.10000coders.in

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It tells a web browser how to display text, images, links, and other elements on a webpage.

What is Bug?

A bug is an error or flaw in software or code that causes it to behave unexpectedly or incorrectly. Bugs can lead to crashes, wrong outputs, or security issues.

Examples:

If a calculator app shows 2 + 2 = 5, that's a bug.

What is an IP Address?

An IP address (Internet Protocol address) is a unique set of numbers assigned to each device connected to the internet. It acts like a digital address so data knows where to go.

Examples:

- IPv4: 192.168.1.1 - IPv6: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

Difference Between Markup and Programming Language?

1. A markup language (like HTML) is used to structure and display content on a webpage. 2. A programming language (like Python) gives the computer instructions to perform tasks. 3. Markup shows how things look, but doesn't "do" anything. 4. Programming makes things happen, like calculations or actions.
